# Single CSS Dark Mode

This example demonstrates how to implement dark mode using a single CSS file with CSS variables and a JavaScript toggle. It avoids separate CSS files for performance and maintainability.

## Language

`html`

## How to Run

Save the code as an HTML file (e.g., index.html) and open it in your web browser.
